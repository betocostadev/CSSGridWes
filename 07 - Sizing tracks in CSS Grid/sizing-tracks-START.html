É possível utilizar porcentagens no sistema GRID sem problemas, porém, não é recomendado já que será necessário fazer alguns cálculos caso você precise utilizar o grip-gap, por exemplo.

Devido a esse "problema" no GRID é recomendado utilizar fr, ou Fraction Units, pois com o fr o sistema já faz o cálculo automaticamente.

Veja a diferença: O GRID superior usa fr para as colunas, já o GRID inferior usa porcentagens.

Também é possível fazer com que os itens do GRID ocupem mais espaço utilizando fr. Se quisermos que, por exemplo, o GRID tenha 4 colunas e as duas do meio sejam maiores, podemos utilizar o código da seguinte maneira:

grid-template-columns: 1fr 2fr 2fr 1fr

Dessa forma, as colunas da direita e esquerda vão ocupar 1fr, enquanto as colunas centrais vão ocupar 2fr cada.
