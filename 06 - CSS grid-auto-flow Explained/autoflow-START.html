O sistema de Grid automaticamente adiciona novas linhas quando precisa para exibir os itens inclusos no código. Caso exista a necessidade de fazer com que o GRID crie novas colunas ao invés de linhas, podemos utilizar um comando para isso.

Por padrão o sistema grid usa grid-auto-flow: row; Podemos mudar para column para que funcione adicionando novas colunas ao invés de linhas.

Este não é um comportamento comum, porém pode ser utilizado caso, por exemplo, seja necessário criar uma galeria com scroll horizontal.
