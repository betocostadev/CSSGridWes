Linhas explícitas são criadas quando definimos tanto linhas quanto as colunas no código.

Por exemplo:

grid-template-columns: 200px 300px;
grid-template-rows: 150px, 150px;

O grid será criado com 2 colunas, uma de 200 px e outra de 300px e terá duas linhas de 150px cara. Caso seja necessário criar uma terceira linha, então o grid irá criar uma linha implícita.

As colunas implícitas podem ter seu tamanho alterado usando o código "grid-auto-rows".
